{\bfseries{Météo Info}} est un site web interactif qui permet de consulter les prévisions météo en France, par ville, département ou région. ~\newline
 Il intègre des données issues d\textquotesingle{}API, et propose une expérience utilisateur personnalisée grâce aux cookies.\hypertarget{md_readme_autotoc_md2}{}\doxysection{\texorpdfstring{🔧 Technologies utilisées}{🔧 Technologies utilisées}}\label{md_readme_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{HTML5}} \+: structure du site
\item {\bfseries{CSS3}} \+: mise en forme (style)
\item {\bfseries{PHP8}} \+: génération dynamique du contenu, enregistrement des consultations
\item {\bfseries{Cookies}} \+: gestion des préférences utilisateur (mode jour/nuit, ville consultée)
\item {\bfseries{APIs}} \+: récupération des données météo au format utilisé
